Task Timer

In this Task Timer module users can start and stop timers to ensure accurate time tracking. It also allows users to set timer limit, the timer stops automatically once the limit is crossed and an email notification is sent to the user. It automatically generates a timesheet when the timer is stopped.

By this module you can eliminate the manual tracking errors and it will improves efficiency of users.

Latest Features

Precise Time Tracking

The module offers precise tracking of task durations, by providing an accurate overview of how much time has been spent on each task.

Automatic Timesheet Recording

Once the timer finishes, the module automatically records the time spent on the task, ensuring accurate and error-free tracking.

Email Notifications

Users receive an email notification when the timer runs out, keeping them informed and alert.

Manual Timer Control

Users have the flexibility to start and end timers manually, providing control over time tracking and ensuring accuracy in time management.

Predefined Timer Description

You can create a Predefined Timer Description that can be selected when you are ending the timer.

Allow Multiple Users to Start Task Timer Simultaneously

By ticking the box of "Allow Multiple User" you allow multiple users to start the task timer simultaneously at once.

Set Timer Task

You have the option of setting the timer for the task. Once you enter the time in Task Timer field, the timer will go off once it reaches the limit.
An Automatic mail will be sent to the user, when the timer ends.

Email Notification

An email will be sent to the user when the timer expires, serving as a notification upon completion

Predefined Timer Description

Select the configuration and choose the Task Timer Description from the dropdown to create a custom timer description.

Click on New to create a new timer description. This will be required when closing the Task Timer under the "Select an option" section

Timer Buttons in Task Kanban View

Kanban view includes both Start and Stop Timer buttons. You can start and stop the Timer directly from My Tasks list.

Timer Button On Task

You can also start the timer directly from the task. Simply select the task, and you'll see the 'Start Task Timer' button in the top-left corner.

Timer is running, you can Stop the timer by clicking on Stop Timer Button.

This wizard will be displayed when you click on 'Stop Timer.' You can also set the start and end times manually by modifying the duration, so that there is an accurate and fixed timesheet for better tracking.

When ending the task timer, the wizard will provide two options: one to simply stop the timer, and the other to mark the task as done while stopping the timer.

Timesheet

Timesheet will also generate automatically when you stop the timer. In the Timesheet you will get the detailed information related to that Task and the actions performed on that duration.

Once the timer expires, a timesheet will be generated with the task description upon completion.

What is the purpose of the Task Timer?
The Task Timer allows users to start and end a timer for a specific task, tracking the start and end times to create timesheets.
Can multiple users use the timer for the same task?
Yes, multiple users can start their own timers for the same task, making it easy to track contributions from various team members.  
How does the auto end feature work?
The auto end feature automatically stops the timer once a predefined duration is reached, as specified in the task's settings.
What happens when I click the End Time button?
Clicking the End Time button opens a wizard where the timer ends, and the timesheet is created, showing the task's start and end times.
Can I manually adjust the start or end times?
The module automatically records the start and end times, but depending on your setup, there may be options to adjust times manually in the timesheet after the task is completed.
